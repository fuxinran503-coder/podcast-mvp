# PodSnap ⚡️

一个为播客听众（特别是驾驶场景）打造的实时转译和灵感记录工具。

## 功能特性

- 🔗 **全网链接导入**：支持小宇宙等平台的播客链接直接导入
- 📁 **本地音频**：支持上传 MP3、WAV、M4A 格式的本地音频文件
- 🎙️ **实时 AI 转写**：使用 OpenAI Whisper 模型将音频实时转换为中文字幕
- 📝 **一键记录**：听到精彩内容时，点击"记一下"即刻保存金句到灵感库
- ✨ **社交海报生成**：4 种美学风格的海报模板，让金句在朋友圈闪光
- 📚 **书籍笔记导出**：一键导出所有记录的金句为书籍样式的长图
- 🛒 **智能商品推荐**：基于播客内容自动推荐相关好物和书籍

## 快速开始

### 本地运行

1. **克隆或下载项目**
```bash
cd podcast-mvp-python
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行应用**
```bash
streamlit run app.py
```

4. **访问应用**
在浏览器打开 `http://localhost:8501`

### 使用步骤

1. **导入播客**
   - 方式一：粘贴小宇宙等平台的播客分享链接，点击"解析并下载"
   - 方式二：上传本地音频文件（MP3、WAV、M4A）

2. **AI 转写**
   - 点击"开始 AI 转写"按钮
   - 等待 Whisper 模型完成转写（首次会自动下载 ~140MB 模型文件）

3. **记录金句**
   - 播放音频，听到精彩内容时点击"记一下"
   - 金句会保存到右侧的灵感库

4. **生成海报**
   - 点击灵感库中的金句卡片
   - 选择喜爱的海报风格（现代、赛博朋克、禅意、复古）
   - 点击"保存海报"下载

5. **导出笔记**
   - 点击灵感库顶部的"导出全部"
   - 系统生成书籍样式的长图
   - 点击"分享笔记长图"下载

## 技术栈

- **前端框架**：Streamlit + 原生 HTML/CSS/JavaScript
- **音频处理**：FFmpeg + imageio-ffmpeg
- **AI 转写**：OpenAI Whisper（base 模型）
- **图片导出**：html2canvas

## 系统要求

- Python 3.8+
- 4GB+ RAM（推荐 8GB）
- 稳定网络连接（首次下载 Whisper 模型）

## 注意事项

- 首次运行会自动下载 Whisper base 模型（~140MB），需要网络连接
- 音频转写时间取决于音频长度和系统性能
- 建议音频文件不超过 100MB，以获得最佳体验
- 所有数据都在本地处理，不上传到服务器
